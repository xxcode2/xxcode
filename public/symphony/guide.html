Symphony Node Installation Guide

Step-by-step guide to set up your Symphony validator node

System Requirements

  • 1

    Hardware Requirements

    4+ CPU cores, 16GB RAM, 500GB SSD storage

  • 2

    Operating System

    Ubuntu 20.04 LTS or newer

  • 3

    Network

    Stable internet connection with static IP

Installation Steps

1. System Preparation

# Update system
sudo apt update && sudo apt upgrade -y

# Install dependencies
sudo apt install -y build-essential git curl wget jq

2. Install Go

# Install Go 1.20
wget https://go.dev/dl/go1.20.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.20.linux-amd64.tar.gz
rm go1.20.linux-amd64.tar.gz

# Add Go to PATH
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc

3. Install Symphony

# Clone repository
git clone https://github.com/symphony-network/symphony.git
cd symphony

# Build binary
make build

# Move binary to system path
sudo mv build/symphonyd /usr/local/bin/

4. Initialize Node

# Initialize node
symphonyd init your-moniker --chain-id symphony-1

# Download genesis file
curl -s https://raw.githubusercontent.com/symphony-network/mainnet/main/genesis.json > ~/.symphony/config/genesis.json

# Set minimum gas price
sed -i 's/minimum-gas-prices = ""/minimum-gas-prices = "0.0001usymph"/' ~/.symphony/config/app.toml

5. Configure Node

# Add persistent peers
sed -i 's/persistent_peers = ""/persistent_peers = "node1.symphony.network:26656,node2.symphony.network:26656"/' ~/.symphony/config/config.toml

# Set pruning
sed -i 's/pruning = "default"/pruning = "custom"/' ~/.symphony/config/app.toml
sed -i 's/pruning-keep-recent = "0"/pruning-keep-recent = "100"/' ~/.symphony/config/app.toml
sed -i 's/pruning-keep-every = "0"/pruning-keep-every = "0"/' ~/.symphony/config/app.toml
sed -i 's/pruning-interval = "0"/pruning-interval = "10"/' ~/.symphony/config/app.toml

6. Create Service

# Create service file
sudo tee /etc/systemd/system/symphonyd.service > /dev/null << EOF
[Unit]
Description=Symphony Node
After=network.target

[Service]
Type=simple
User=$USER
WorkingDirectory=$HOME
ExecStart=/usr/local/bin/symphonyd start
Restart=on-failure
RestartSec=10
LimitNOFILE=65535

[Install]
WantedBy=multi-user.target
EOF

# Start service
sudo systemctl daemon-reload
sudo systemctl enable symphonyd
sudo systemctl start symphonyd

Important Notes

1. Replace 'your-moniker' with your desired validator name

2. Make sure to backup your validator keys

3. Wait for the node to sync before creating validator

Useful Commands

# Check node status
symphonyd status

# Check sync status
symphonyd status 2>&1 | jq .SyncInfo

# Check validator status
symphonyd query staking validator $(symphonyd keys show wallet --bech val -a)
# View logs
sudo journalctl -u symphonyd -f

# Restart node
sudo systemctl restart symphonyd

# Stop node
sudo systemctl stop symphonyd
