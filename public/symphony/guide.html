Symphony Setup Guide

Complete guide to setting up and running a Symphony node

System Requirements

Component Minimum Recommended
CPU 4 cores 8+ cores
RAM 8GB 16GB+
Storage 500GB SSD 1TB+ NVMe SSD
Network 100 Mbps 1 Gbps

Installation Steps

1

Update System

First, update your system packages to ensure you have the latest security patches and dependencies.

sudo apt update && sudo apt upgrade -y
2

Install Dependencies

Install required dependencies for running a Symphony node.

sudo apt install -y build-essential git curl wget jq
3

Install Go

Install Go programming language required for Symphony.

wget https://go.dev/dl/go1.21.0.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.21.0.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
4

Install Symphony

Clone and build the Symphony repository.

git clone https://github.com/symphony-network/symphony.git
cd symphony
make install
5

Initialize Node

Initialize your Symphony node with a custom moniker.

symphonyd init "your-moniker" --chain-id symphony-1
6

Configure Node

Configure your node settings and download the genesis file.

curl -s https://raw.githubusercontent.com/symphony-network/mainnet/main/genesis.json > ~/.symphony/config/genesis.json
7

Create Service

Create a systemd service for your Symphony node.

sudo tee /etc/systemd/system/symphony.service > /dev/null << EOF
[Unit]
Description=Symphony Node
After=network.target

[Service]
Type=simple
User=$USER
WorkingDirectory=$HOME
ExecStart=$(which symphonyd) start
Restart=on-failure
RestartSec=10
LimitNOFILE=65535

[Install]
WantedBy=multi-user.target
EOF
8

Start Node

Start your Symphony node service.

sudo systemctl daemon-reload
sudo systemctl enable symphony
sudo systemctl start symphony

Useful Commands

Check Node Status

symphonyd status

View Logs

sudo journalctl -u symphony -f

Important Notes

⚠️ Security Considerations

Always keep your validator keys secure and never share them with anyone. Consider using a hardware wallet for additional security.

💡 Performance Tips

For optimal performance, ensure your node has sufficient resources and a stable internet connection. Regular maintenance and updates are recommended.

Troubleshooting

Common Issues

Node Not Syncing

If your node is not syncing properly, try these steps:

  • Check your internet connection
  • Verify system requirements are met
  • Check node logs for errors
  • Try resetting the node

High Resource Usage

If your node is using too many resources:

  • Check for background processes
  • Optimize system settings
  • Consider upgrading hardware

Frequently Asked Questions

How long does it take to sync?

Initial sync time depends on your hardware and internet connection. Typically, it takes 24-48 hours with recommended specifications.

What are the minimum staking requirements?

The minimum staking amount is 1,000 SYMPHONY tokens. However, higher amounts increase your chances of being selected as a validator.

How do I update my node?

To update your node, follow these steps:

# Stop the node
sudo systemctl stop symphony

# Update the binary
cd symphony
git pull
make install

# Start the node
sudo systemctl start symphony
