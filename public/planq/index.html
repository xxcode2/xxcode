Planq Node Installation Guide

A comprehensive guide to installing and configuring a Planq blockchain node on your server. Follow these steps to set up your node quickly and efficiently.

System Requirements

  • 4 CPU cores
  • 8GB RAM
  • 200GB SSD storage
  • Stable internet connection

Estimated Time

  • Setup: 15-20 minutes
  • Sync: 2-3 hours
  • Validation: 10 minutes

Useful Links

  • Official Documentation
  • GitHub Repository
  • Discord Community
  • Network Explorer

Installation Steps

1

Update System Packages

First, make sure your system is up to date with the latest packages.

sudo apt update && sudo apt upgrade -y
Copied!
2

Install Dependencies

Install the necessary dependencies for the Planq node software.

sudo apt install curl build-essential git wget jq make gcc tmux htop nvme-cli pkg-config libssl-dev libleveldb-dev tar clang bsdmainutils ncdu unzip lz4 -y
Copied!
3

Install Go

Install Go programming language which is required for building the Planq node.

wget https://go.dev/dl/go1.20.5.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.20.5.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
Copied!
4

Clone Planq Repository

Clone the official Planq repository to your local machine.

git clone https://github.com/planq-network/planq.git
cd planq
Copied!
5

Build Planq

Build the Planq node software from source.

make install
Copied!

Configuration

planqd init mynode --chain-id planq_7000-1
planqd config chain-id planq_7000-1
Copied!

Start Node

planqd start
Copied!

Frequently Asked Questions

The minimum requirements are 4GB RAM, 100GB storage, and a stable internet connection. For optimal performance, we recommend 8GB RAM and 200GB storage.
Initial sync time depends on your internet connection and hardware. Typically, it takes 24-48 hours to fully sync with the network.
To update your node, stop the current process, pull the latest changes from the repository, rebuild, and restart the node.
