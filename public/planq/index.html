Planq Node Installation Guide

Introduction

Planq is a high-performance blockchain platform that enables developers to build and deploy decentralized applications. This guide will help you set up a Planq node on your system.

Prerequisites

  • Ubuntu 20.04 or later
  • Minimum 4GB RAM
  • 100GB free disk space
  • Stable internet connection

Installation Steps

1

Update System

sudo apt update && sudo apt upgrade -y
Copied!
2

Install Dependencies

sudo apt install -y build-essential git curl wget
Copied!
3

Install Go

wget https://go.dev/dl/go1.20.5.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.20.5.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
Copied!
4

Clone Planq Repository

git clone https://github.com/planq-network/planq.git
cd planq
Copied!
5

Build Planq

make install
Copied!

Configuration

planqd init mynode --chain-id planq_7000-1
planqd config chain-id planq_7000-1
Copied!

Start Node

planqd start
Copied!

Frequently Asked Questions

The minimum requirements are 4GB RAM, 100GB storage, and a stable internet connection. For optimal performance, we recommend 8GB RAM and 200GB storage.
Initial sync time depends on your internet connection and hardware. Typically, it takes 24-48 hours to fully sync with the network.
To update your node, stop the current process, pull the latest changes from the repository, rebuild, and restart the node.
