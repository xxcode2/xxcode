Planq Node Installation Guide

A comprehensive guide to installing and configuring a Planq blockchain node on your server. Follow these steps to set up your node quickly and efficiently.

System Requirements

  • 4 CPU cores
  • 8GB RAM
  • 200GB SSD storage
  • Stable internet connection

Estimated Time

  • Setup: 15-20 minutes
  • Sync: 2-3 hours
  • Validation: 10 minutes

Useful Links

  • Official Documentation
  • GitHub Repository
  • Discord Community
  • Network Explorer

Installation Steps

1

Update System Packages

First, make sure your system is up to date with the latest packages.

sudo apt update && sudo apt upgrade -y
Copied!
2

Install Dependencies

Install the necessary dependencies for the Planq node software.

sudo apt install curl build-essential git wget jq make gcc tmux htop nvme-cli pkg-config libssl-dev libleveldb-dev tar clang bsdmainutils ncdu unzip lz4 -y
Copied!
3

Install Go

Install Go programming language which is required for the Planq node software.

cd $HOME
ver="1.19.5"
wget "https://golang.org/dl/go$ver.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go$ver.linux-amd64.tar.gz"
rm "go$ver.linux-amd64.tar.gz"
Copied!
4

Setup Environment Variables

Configure your environment variables for Go and the Planq node software.

echo "export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin" >> $HOME/.bash_profile
echo "export PLANQ_CHAIN_ID=planq_7070-2" >> $HOME/.bash_profile
echo "export PLANQ_MONIKER=your-node-name" >> $HOME/.bash_profile
source $HOME/.bash_profile
Copied!

Make sure to replace your-node-name with your desired node name.

5

Clone and Build the Planq Node Software

Download and compile the Planq node software from the source.

cd $HOME
git clone https://github.com/planq-network/planq.git
cd planq
git checkout v1.0.7
make install
Copied!
6

Initialize the Node

Initialize your Planq node with the chain ID.

planqd init $PLANQ_MONIKER --chain-id $PLANQ_CHAIN_ID
Copied!
7

Download Genesis File

Download the genesis file for the Planq network.

wget -O $HOME/.planqd/config/genesis.json "https://raw.githubusercontent.com/planq-network/networks/main/mainnet/genesis.json"
Copied!

9

Configure Pruning

Configure pruning settings to optimize disk space usage.

pruning="custom"
pruning_keep_recent="100"
pruning_keep_every="0"
pruning_interval="50"

sed -i -e "s/^pruning *=.*/pruning = \"$pruning\"/" $HOME/.planqd/config/app.toml
sed -i -e "s/^pruning-keep-recent *=.*/pruning-keep-recent = \"$pruning_keep_recent\"/" $HOME/.planqd/config/app.toml
sed -i -e "s/^pruning-keep-every *=.*/pruning-keep-every = \"$pruning_keep_every\"/" $HOME/.planqd/config/app.toml
sed -i -e "s/^pruning-interval *=.*/pruning-interval = \"$pruning_interval\"/" $HOME/.planqd/config/app.toml
Copied!
10

Create Service File

Create a systemd service file to manage your Planq node.

sudo tee /etc/systemd/system/planqd.service > /dev/null << EOF
[Unit]
Description=Planq Node
After=network-online.target

[Service]
User=$USER
ExecStart=$(which planqd) start
Restart=always
RestartSec=3
LimitNOFILE=65535

[Install]
WantedBy=multi-user.target
EOF
Copied!
11

Start the Node

Enable and start your Planq node service.

sudo systemctl daemon-reload
sudo systemctl enable planqd
sudo systemctl start planqd
Copied!

Frequently Asked Questions

The initial sync time depends on network conditions and your hardware. Typically, it takes 2-3 hours for a full sync with the recommended hardware specifications. You can check the sync status using the command provided in the Useful Commands section.

Common issues can be resolved by:

  • Ensuring your system meets the minimum requirements
  • Checking logs for specific error messages using sudo journalctl -u planqd -f -o cat
  • Verifying network connectivity to peers
  • Joining the Planq Discord community for support

To update your Planq node to a newer version:

  1. Stop the node service: sudo systemctl stop planqd
  2. Go to the repository directory: cd $HOME/planq
  3. Fetch the latest changes: git fetch
  4. Checkout the new version: git checkout [new-version]
  5. Rebuild: make install
  6. Restart the service: sudo systemctl start planqd

To become a validator on the Planq network:

  1. Ensure your node is fully synced
  2. Create a wallet and obtain PLANQ tokens
  3. Create your validator with the following command:
planqd tx staking create-validator \
--amount=1000000000000aplanq \
--pubkey=$(planqd tendermint show-validator) \
--moniker="YOUR_VALIDATOR_NAME" \
--identity=YOUR_KEYBASE_ID \
--website="YOUR_WEBSITE" \
--details="YOUR_DESCRIPTION" \
--chain-id=planq_7070-2 \
--commission-rate=0.10 \
--commission-max-rate=0.20 \
--commission-max-change-rate=0.01 \
--min-self-delegation=1 \
--from=wallet \
--gas-prices=30000000000aplanq \
--gas=auto \
--gas-adjustment=1.5

Replace the placeholder values with your own information.
